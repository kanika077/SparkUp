Spark-Up – Disaster Education & Management System

Spark-Up is a web-based Disaster Education and Emergency Response prototype developed for Smart India Hackathon (SIH).
The platform helps users learn disaster preparedness through interactive modules, quizzes, simulations, and emergency tools such as shelter location mapping and panic alerts.

The system aims to improve awareness, preparedness, and quick response during disasters.

Features
Learning Modules – Educational content on earthquake, flood, and fire safety.
Knowledge Base – Guides on emergency kits, first aid, and disaster awareness.
Quiz System – Test user understanding of disaster management concepts.
Survival Simulation Game – Interactive practice for emergency decision-making.
Interactive Map – Displays safe shelters using Leaflet.js and OpenStreetMap.
Locate Me – Shows the user's real-time location using Geolocation API.
Panic Alert – Simulated emergency alert with location sharing
Teacher Dashboard – Allows teachers to monitor student progress and send alerts.
Progress Tracking – Tracks modules, quiz scores, and badges.

Technologies Used
HTML5
CSS3
JavaScript
Leaflet.js
OpenStreetMap
Browser Geolocation API
LocalStorage

Objective
The objective of this project is to enhance disaster awareness and preparedness by providing an interactive platform that combines learning resources with emergency response tools.
